Описание программного комплекса для определения прочностных характеристик горной породы

 

Программа для ЭВМ «Определение геомеханических свойств мультислойных горных пород в сложных напряженных состояниях» позволяет определить разрушающие напряжения горной породы, с учетом проведении минимального количества испытаний на одноосное сжатие под разными углами относительно оси полноразмерного керна. Определяются прочностные характеристики при сдвиге вдоль и поперек слоев в виде сцепления и угла внутреннего трения. Также из линейного критерия прочности для анизотропных горных пород, по программе вычисляются значения предельных напряжений при объемном сжатии под различными углами к направлению слоистости. Исходными параметрами при этом принимаются пределы прочности, сцепление и угол внутреннего трения в разных направлениях слоистости.

Программа предназначена для решения геомеханических задач в области бурения и разработки нефтяных и газовых месторождений, при проектировании бурения скважин в сложном геологическом разрезе, а также при решении задач интенсификации добычи в сложнопостроенных залежах углеводородов.

Тип ЭВМ: 

Язык: C#

Фреймворк: WPF

ОС: Microsoft Windows 8-11

Объём программы: 137 Mb

После запуска программы открывается окно 1 показанное на рисунке 1. Слева на странице расположены поля для ввода входных данных, справа подбираемые переменные и результаты вычислений. Это окно отвечает за подбор необходимого параметра прочности (SIG1). Для этого нам необходимо ввести исходные данные, такие как:

С0 – сцепление горной породы образца керна, выбуренной при 00 относительно напластования;

С90 – сцепление горной породы образца керна, выбуренной при 900 относительно напластования;

ro0 – угол внутреннего трения горной породы образца керна, выбуренной при 00 относительно напластования;

ro90 – угол внутреннего трения горной породы образца керна, выбуренной при 900 относительно напластования;

SIG3 – боковое давление горной породы, МПа;

betta – угол, под которым был вырезан образец.

Рисунок 1 – Окно приложения

При нажатии на кнопку «Подобрать» программа начнет выполнять алгоритм подбора, после чего обновит значения в соответствующих полях. В результате отразятся значения, такие как:

SIG1 – максимальное (главное) разрушающее напряжение горной породы, МПа;

PSI – поправочный коэффициент, варьируется в пределах от 2,9 при 00 до 0,4 при 900.

F1 – формула (1) из отчета по 1 этапу, при значении близкого к нулю, доказывающая правильность подбора SIG1.

F2 - формула, при значении близкого к нулю, доказывающая правильность подбора Psi.

При нажатии на кнопку «Вычислить» программа вычисляет значения.

При активном состояние флажка «Расчёт таблицы» алгоритм будет подбирать значения с использованием заданного пользователем набора входных данных.

При нажатии на кнопку «Таблица» откроется окно для настройки набора входных значений. На рисунке 2 показано окно настройки набора значений.

Рисунок 2 – Настройка набора значений

Кнопка «+» добавляет новое значение в набор, которое было задано пользователем в поле. Кнопка «-» удаляет выбранное значение из набора данных. Кнопка «Очистить» полностью очищает список.

На рисунке 3 показан результат выполнения программы «Расчёт таблицы».

Рисунок 3 – Результат выполнения

Алгоритм подобрал значения SIG1 и PSI для получения значений F1 и F2 таким образом, чтобы расстояние между числами была минимально. Также по окончанию работы алгоритм выводится отчёт об вычисленных наборов значений (таблицы). В данном случае ro0 не является переменной, чтобы задать ее как переменную, нужно в раскрывающимся списке значение списка с «SIG1» на «ro0». Данная интерпретация необходима для уточнения ro0, если на руках имеются эксперименты на трехосное сжатие.

В строке указаны первые две подобранные переменные, а затем и результат. Если пользователь уберёт флажок с «Расчёт таблицы», то табличный расчёт производится не будет. При нажатии на кнопку «Экспорт в Excel» программа создаст Excel файл и перенесет все значения в файл.

Над вкладками раздела расчётов располагаются кнопки с изображениями.

Первая кнопка позволяет переносить входные данные в другие разделы расчётов, позволяя заново не вводить уже заданные значения. Вторая кнопка позволяет включить или отключить всплывающее окно.

На рисунке 4 показан 2 раздел расчётов.

Рисунок 4 – 2 раздел расчёта

Слева на странице расположены поля для ввода данных экспериментов, справа подбираемые переменные и результаты вычислений. Второй раздел отвечает за подбор необходимых параметров, таких как (С0, С90, ro0, ro90), которые служат исходными данными для работы в первом разделе.

Для вычисления этих данных (расположенных справа), необходимо задать следующие значения:

S11 – прочностные характеристики, полученные экспериментально, первого из трех образцов, МПа;

S13 – боковое давление, действующие на первый образец, МПа;

S21 – прочностные характеристики, полученные экспериментально, второго из трех образцов, МПа;

S23 – боковое давление, действующие на второй образец, МПа;

S31 – прочностные характеристики, полученные экспериментально, третьего из трех образцов, МПа;

S33 – боковое давление, действующие на третий образец, МПа;

 α1,α2, α3 – углы выбуривания образца относительно напластования.

Подбирает такое значение C0, чтобы значение F было близко к 0.

На рисунке 5 показан 3 раздел расчёта.

Рисунок 5 – 3 раздел расчёта

Третий раздел также служит для определения таких параметров как С0, С90, ro90 при наличии экспериментов на трехосное сжатие. Для нахождения ro0, полученные данные переносим в первый раздел, где программа подбирает значение. Правильность подобранных значений подтверждается, когда значение Z1 и Z2 между собой равны.
